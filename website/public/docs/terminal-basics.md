## A Brief History: From DOS to the Modern Shell

Before computers had mice and colorful windows (the "Graphical User Interface" or GUI), we talked to them through text.

- **The DOS Era:** In the 1980s, PC users used **MS-DOS**. You had to type everything you wanted the computer to do.
- **The Unix Revolution:** At the same time, powerful systems like Unix (which modern Mac and Linux are based on) were developing the "Shell"â€”a layer that surrounds the computer's core and lets you interact with it.
- **Why It Still Exists:** GUIs are great for discovery, but the terminal is built for **speed** and **automation**. While a GUI lets you click a button, a terminal lets you write a script that clicks that button 10,000 times in a second.

## Under the Hood: The "Vibe" Logic

When you use Antigravity, it's often running terminal commands for you. It's essentially "vibecoding" by talking to the computer's oldest, most reliable interface.

### The Terminal is a Programming Environment
The terminal isn't just a place to run programs; it's a program itself. On Mac and Linux, the most common shell is **bash** (or **zsh**). On Windows, it's **PowerShell**.
These are full programming languages. They can:
- Remember information (variables)
- Make decisions (if/then)
- Repeat tasks (loops)

## Core Commands: The Survival Kit

Even with AI doing the heavy lifting, knowing these core commands helps you understand what's happening "under the hood."

| Command | What it does | Real-world equivalent |
| :--- | :--- | :--- |
| `ls` (Mac) / `dir` (Win) | **List** files in the current folder | Opening a folder in Finder/Explorer |
| `cd` | **Change Directory** (move to a folder) | Double-clicking a folder |
| `mkdir` | **Make Directory** (create a folder) | Right-click -> New Folder |
| `cp` | **Copy** a file or folder | Ctrl+C / Ctrl+V |
| `mv` | **Move** or rename a file | Dragging a file to a new spot |
| `rm` | **Remove** (delete) a file | Moving to Trash (but permanent!) |

## Why Use the CLI (Command Line Interface)?

### 1. No Repetitive Clicking
Imagine you need to rename 500 photos to include the date. In a GUI, that's an hour of clicking. In the terminal, it's one line of code.

### 2. Full Execution Environment
The terminal can run **CLI Tools**. These are specialized programs without a "face" (no windows or buttons) that do one thing extremely wellâ€”like converting a video, resizing an image, or deploying a website to the internet.

### 3. The Power of "Piping"
You can take the output of one command and "pipe" it into another.
> *Example: "Find all files larger than 100MB* **AND** *move them to the archive folder."*
